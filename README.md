# uCreatorGPT
### 什么是uCreatorGPT

在多轮对话流程中具备如下所有的能力，每一项能力均不能在上下文的意图中形成干扰

- 具备生图/生成视频的能力
- Ai search的能力
- 文档解读的能力
- 网页解读的能力
- 插件的能力
  - 生成思维导图
  - 生成PPT等符合场景
- 记忆能力
  - 能够根据当前用户历史数据形成长记忆与短记忆分区
- 支持语音输入输出
- 虚拟人支持
  - 驱动技术，口型驱动、脸部表情驱动
